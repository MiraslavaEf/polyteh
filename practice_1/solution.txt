Ответы на вопросы:
GitHub — это платформа для хостинга IT-проектов. Это как
облачное хранилище для вашего кода, где вы можете:
● Делать резервные копии. Ваш код хранится на сервере, и вы не
потеряете его, даже если что-то случится с компьютером.
● Работать в команде. GitHub упрощает совместную работу над
проектами: вы можете видеть изменения других разработчиков,
оставлять комментарии и предлагать свои правки.
● Делать проекты публичными. GitHub позволяет поделиться
своим проектом с миром.
Repository (Репозиторий) — место хранения ваших файлов на
GitHub.
В контексте системы контроля версий с помощью гитхаба можно делать бекапы для отката проекта к предыдущей версии
Приватный репозиторий видим только мы и те, кого мы добавили
Ветвление используется для отдельной работы, без затрагивания основных файлов. Для работы с отдельными частями проекта.
Главная ветка (main, ранее master) в Git предназначена для хранения стабильного, протестированного кода, готового к развертыванию в рабочей среде (продакшене).
